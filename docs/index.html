TigerNav: Development of a Virtual Assistant using an Autoregressive Model for Indoor Navigation

A dialogue-based indoor navigation system using a Large Language Model to process diverse user inputs and provide accurate directional guidance through voice and text interaction.

üìÑ Abstract

We present TigerNav, a dialogue-based indoor navigation assistant that uses Large Language Models to provide interactive and speech-enabled directional guidance. Addressing challenges users faced with navigating complex indoor environments using conventional navigation systems, TigerNav integrates Whisper speech-to-text and Coqui text-to-speech technologies with a fine-tuned GPT-2 model, enabling natural voice-based and text-based interaction for indoor navigation. Deployed specifically for the Fr. Roque Rua√±o Building at the University of Santo Tomas Engineering campus, the system was trained on a custom dataset designed to handle real-world navigation scenarios with both formal and informal user queries. The system supports both voice and text modalities for flexible interaction. We conclude with performance benchmarks, user satisfaction analysis, and recommendations for future enhancements. The source code can be accessed via https://github.com/tigernav-ust/tigernav.

TigerNav employs a three-component pipeline: users speak navigation queries through Whisper STT, the fine-tuned GPT-2 model processes the request and generates directional instructions, and Coqui TTS delivers the response through natural speech output.

üé§ Voice-Enabled Virtual Assistant

Users can speak naturally to TigerNav and receive voice-guided navigation instructions, enabling hands-free operation ideal for walking through buildings.

üí¨ Text-Based Chatbot

For quieter environments or user preference, TigerNav also provides a text-based chat interface with the same navigation capabilities.
